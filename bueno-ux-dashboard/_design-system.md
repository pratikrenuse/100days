# Bueno — Design System & UX reference

Reusable notes for any Bueno product (apply alongside the **clear-ui** skill).

## Brand (from the official Bueno identity guidelines)

**Logo / wordmark.** "Bueno" — a high-contrast serif set in **small-caps** (large cap `B`, small-cap `UENO`). Payoff: *Property · Banking · Simplified*. Negative versions: white `#FFFFFF`, light grey `#D9D9D9`, pale blue `#CBE8FF` on dark.

**Colour — core**
| Role | Hex |
|---|---|
| Core navy (logo, headlines, body) | `#010221` |
| Secondary navy | `#292A49` |

**Colour — backgrounds:** pale blue `#CBE8FF`, light grey `#D9D9D9`, white `#FFFFFF`, off-white `#F8F7F4`.

**Colour — accents (use restrained):** teal `#0B7374`, bright teal `#00BFAF`, rust `#C43402`, deep maroon `#390D02`, olive `#4B4633`, sage `#BEBE9C`, warm yellow `#FCC662`. "In the initial phase the use of these will be restrained and careful."

**Type.** Corporate font: **FS Siena** (a humanist *sans-serif*) for headings and body. Web fallback: Jost. The serif is **logo-only** (web fallback: Cormorant Garamond, `font-variant: small-caps`).

## CLEAR UI checklist (condensed — full skill: `clear-ui`)

1. **Copywriting** — buttons are `verb + object` (never "Submit/Next/OK"); hero answers "what's in it for me" in one line; reassure at friction points.
2. **Layout** — group with whitespace + common-region, not borders; card padding ≥2rem; one focus per section.
3. **Emphasis** — exactly ONE primary action per section; turn up a single dial (size/colour/space); accent colour used sparingly; pass the Squint Test.
4. **Accessibility** — 44px+ targets; body contrast ≥4.5:1 (target 7:1); never colour-only meaning; primary action above the fold.
5. **Reward** — every completed action gets a specific confirmation; for a finance product, prefer **Control** rewards ("Saved · encrypted · you can undo").

## How this dashboard applies it
- Navy `#010221` panels, off-white `#F7F6F2` page, white cards; **teal** as the single accent; rust/yellow only in small status pills.
- High-contrast light text on navy (no thin low-opacity text).
- One headline metric leads the executive brief; charts carry the data relationships (Emphasis → Visualization).
- Status uses colour **plus** a label/pill (not colour alone).
